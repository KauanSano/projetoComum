Introdução a Banco de Dados

Um banco de dados é um sistema organizado de informações estruturadas que podem ser armazenadas, gerenciadas e acessadas de forma eficiente. São amplamente usados em diferentes setores.

Introdução a MER

Modelo Entidade-Relacionamento (MER) é uma abordagem de modelagem de dados que representa as entidades relevantes em um sistema, seus atributos e relacionamentos.

Modelo Relacional

O modelo relacional é uma abordagem para o design de bancos de dados que organiza os dados em tabelas, com linhas representando registros e colunas representando atributos.

SQL

SQL (Structured Query Language) é uma linguagem de consulta estruturada usada para gerenciar bancos de dados relacionais e executar operações neles. Com o intuito de incluir, excluir, alterar dados, etc.
