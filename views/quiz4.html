Comandos básicos

1. Qual comando SQL é usado para criar uma nova tabela em um banco de dados?

a) INSERT
b) UPDATE
c) DELETE
d) CREATE

2. Qual comando SQL é usado para selecionar dados de uma tabela?

a) SELECT
b) WHERE
c) FROM
d) ORDER BY

3. Qual comando SQL é usado para atualizar registros em uma tabela?

a) SELECT
b) INSERT
c) UPDATE
d) UPDATE

4. Qual comando SQL é usado para excluir registros de uma tabela?

a) DELETE
b) DROP
c) TRUNCATE
d) REMOVE

5. Qual comando SQL é usado para inserir novos registros em uma tabela?

a) INSERT
b) UPDATE
c) ADD
d) CREATE

6. Qual comando SQL é usado para renomear uma tabela?

a) RENAME TABLE
b) ALTER TABLE
c) UPDATE TABLE
d) MODIFY TABLE

7. Qual comando SQL é usado para criar uma chave primária em uma tabela?

a) PRIMARY KEY
b) FOREIGN KEY
c) UNIQUE KEY
d) INDEX

8. Qual comando SQL é usado para adicionar uma nova coluna a uma tabela existente?

a) ALTER TABLE
b) MODIFY TABLE
c) ADD COLUMN
d) UPDATE TABLE

9. Qual comando SQL é usado para realizar uma junção (join) entre duas tabelas?

a) UNION
b) JOIN
c) MERGE
d) COMBINE

10. Qual comando SQL é usado para ordenar os resultados de uma consulta em ordem descendente?

a) ASC
b) DESC
c) ORDER BY
d) SORT BY
