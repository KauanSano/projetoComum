Introdução a MER

1.No contexto do Modelo Entidade-Relacionamento (MER), o que representa uma entidade?

a) Uma propriedade ou característica de um objeto.
b) Um relacionamento entre duas entidades.
c) Um conjunto de objetos semelhantes.
d) Um objeto ou conceito do mundo real.

2. Qual dos seguintes símbolos é usado para representar um relacionamento no diagrama do MER?

a) Círculo
b) Retângulo
c) Losango
d) Triângulo

3. No MER, o que é um atributo?

a) Uma tabela que armazena dados.
b) Um conjunto de entidades relacionadas.
c) Uma descrição de uma relação entre entidades.
d) Uma propriedade ou característica de uma entidade.ATE

4. O que representa uma cardinalidade no diagrama do MER?

a) O número total de entidades em um relacionamento.
b) O tipo de dados usado para um atributo.
c) O número máximo e mínimo de ocorrências de uma entidade em um relacionamento.
d) O nome dado a uma tabela no banco de dados.

5. Qual dos seguintes representa um relacionamento "muitos para muitos" no diagrama do MER?

a) Linha contínua.
b) Linha tracejada.
c) Linha com seta.
d) Linha dupla.

6. No MER, o que é um atributo composto?

a) Um atributo que contém valores de outros atributos.
b) Um atributo que armazena valores nulos.
c) Um atributo que não pode ser dividido em partes menores.
d) Um atributo que representa a chave primária de uma entidade.

7. Qual dos seguintes é um exemplo de restrição de integridade no MER?

a) Um atributo que não pode conter valores duplicados.
b) Um relacionamento entre duas entidades.
c) Um atributo que não pode ser nulo.
d) Uma entidade que não pode ter mais de uma ocorrência.

8. O que é um modelo de entidade fraca no MER?

a) Um modelo que não possui relacionamentos com outras entidades.
b) Um modelo em que todas as entidades possuem atributos compostos.
c) Um modelo em que as entidades não têm identificador único.
d) Um modelo em que as entidades dependem de outra entidade para existir.

9. Qual é o objetivo do diagrama do MER?

a) Representar a estrutura física do banco de dados.
b) Definir as consultas SQL a serem executadas no banco de dados.
c) Ilustrar as relações entre as entidades de um sistema.
d) Armazenar os registros de dados no banco de dados.

10. O que é a generalização/especialização no contexto do MER?

a) Um processo de agrupar entidades semelhantes em categorias mais gerais.
b) Um relacionamento entre entidades com base em seus atributos compartilhados.
c) Uma técnica para criar um atributo composto a partir de outros atributos.
d) Um método para atribuir cardinalidades aos relacionamentos entre entidades.
