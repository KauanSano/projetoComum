Introdução a banco de dados

1. Qual das seguintes opções define corretamente um banco de dados?

a) Uma coleção de tabelas em formato de planilha
b) Um software para criar e gerenciar apresentações de slides.
c) Um sistema que permite armazenar, gerenciar e organizar dados de forma organizada.
d) Uma lista de contatos armazenada em um smartphone.

2. Quais são as principais aplicações de um banco de dados?

a) Cálculos matemáticos complexos.
b) Manipulação de imagens e vídeos.
c) Armazenamento e recuperação eficiente de dados.
d) Execução de tarefas de processamento de texto.

3. Qual das opções abaixo não é um exemplo de SGBD?

a) DataMaster SQL
b) POSTGRESQL
c) MYSQL
d) SQL Server

4. Qual tipo de modelos de dados o POSTGRESQL utiliza?

a) Abordagem hierárquica.
b) Abordagem em rede.
c) Abordagem relacional.
d) Abordagem orientada a objetos.

5. Qual das opções abaixo não é um comando de banco de dados relacional?

a) Criação de banco de dados e estrutura da tabela.
b) Manipulação dos dados para extrair informações.
c) Administração de dados.
d) Duplicação de tabelas.

6. Qual a necessidade da modelagem de um Banco de dados?

a) É necessária para organizar os dados e garantir sua integridade
b) É necessária para facilitar o acesso a dados
c) Necessário para dificultar o desenvolvimento de aplicações
d) Para permitir que os dados sejam privados

7. O que o comando CREATE faz?

a) O comando CREATE cria uma tabela e especifica os dados e os tipos de dados presentes nela.
b) O comando CREATE altera os dados de uma tabela.
c) O comando CREATE exclui os dados de uma tabela.
d) O comando CREATE insere dados em uma tabela.

8. O que o comando INSERT faz?

a) Exclui dados ta tabela.
b) Insere dados na tabela.
c) Altera dados da tabela.
d) Exclui toda a tabela.

9. Qual dos comandos abaixo retorna os dados de uma tabela?

a) SELECT * FROM Alunos.
b) SELECT * FROM.
c) SELECT Alunos.
d) FROM Alunos SELECT *.

10. O que significa SQL?

a) System Query Language.
b) Standard Query Language.
c) Server Query Language.
d) Structured Query Language.
